# 🗺️ Roadmap - План розвитку

## Версія 1.0 (Поточна) ✅

### Функціональність
- ✅ Авторизація з логіном/паролем
- ✅ Профіль користувача
- ✅ Список новин
- ✅ Деталі новини з картинками та посиланнями
- ✅ Голосування (активні та завершені)
- ✅ Один голос на користувача
- ✅ Синя/голуба колірна схема

### Backend
- ✅ Node.js Express сервер
- ✅ MySQL база даних
- ✅ JWT аутентифікація
- ✅ API для авторизації, новин та голосування
- ✅ Валідація даних
