## [1.0.0] - 2024-01-01

### 🎉 Першій релізу!

#### ✨ Додано
- Авторизація з логіном та паролем
- Управління сесією через JWT токени
- Профіль користувача з ПІБ та групою
- Список новин з пошуком та фільтрацією
- Деталі новини з повним вмістом
- Гортання картинок у новинах (carousel)
- Посилання у новинах
- Активні голосування з виборами
- Завершені голосування з результатами
- Один голос на користувача за голосування
- Синя/голуба колірна схема
- Локальне зберігання даних користувача
- Pull-to-refresh для оновлення даних
- Обробка помилок та з'єднання
- Навігація через BottomNavigationBar

#### 🔧 Backend
- Node.js Express сервер
- MySQL база даних
- JWT аутентифікація та авторизація
- API для авторизації
- API для новин
- API для голосувань
- Bcryptjs хешування паролів
- CORS підтримка
- Error handling middleware
- Database connection pooling

## Знані проблеми

### v1.0.0
- ⚠️ Якщо сервер повільний, картинки можуть завантажуватися довго
- ⚠️ На деяких пристроях Android Emulator потребує `10.0.2.2` замість `localhost`
- ⚠️ Rate limiting не реалізований (планується в v1.1)

### v1.0.1
- ✅ Всі мені известні проблеми виправлені
